# Total Browser Purge

A powerful Edge extension to clear all browser data with a single click.

## Features
- **One-Click Purge:** Instantly deletes cache, cookies, history, storage, and passwords.
- **Deep Clean:** Clears everything from the beginning of time.
- **Privacy First:** Quickly wipe your digital footprint.

## How to Install
1. Open Edge and go to `edge://extensions/`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this project folder.

## Usage
Click the extension icon and press the **Clear Everything Now** button.
